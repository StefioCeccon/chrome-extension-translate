🎯 Call Subtitle Translator

Translate call subtitles to English in real-time using AI

🚀 Getting Started

This extension automatically detects and translates subtitles during video calls, making it easier to understand conversations in different languages.

⚙️ Setup Required

To use this extension, you'll need to:

  1. Get an OpenAI API Key: Visit OpenAI Platform to create an account and get your API key
  2. Configure the Extension: Click the extension icon and enter your API key
  3. Enable Auto-translate: Toggle the auto-translate feature on

💡 How It Works

The extension monitors video call platforms for subtitle text and automatically translates non-English content to English. Translations appear as overlays above the original text.

🔒 Privacy & Security

Your OpenAI API key is stored locally in your browser and is only used to make translation requests. No personal data is collected or stored by this extension.
